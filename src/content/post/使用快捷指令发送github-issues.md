## 快捷指令
https://www.icloud.com/shortcuts/1445a4ce30dc4ccbb15e7090b4c3e9ff

- 支持多图，只能放在在文章前面
- 自动转换到JPEG。

## 配置Config

- token 填写你的token
- owner 填写github用户名
- repo 填写仓库名
- src 填写存放图片的位置
- cdn 如无其他cdn保持默认
- api 如无其他api地址保持默认
